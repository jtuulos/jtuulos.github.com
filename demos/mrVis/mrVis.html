Disco (MapReduce) Visualization

This is a very basic prototype visualization of MapReduce. This prototype is part of my bachelor's thesis at Aalto University and is coded entirely in client-side JavaScript/jQuery. The program parses a Disco log file and produces a stepwise visualization from it.

code/contact: Jyri Tuulos - data: http://discoproject.org/ - plumbing: jsPlumb by Simon Porritt

(press 'j' to advance the visualization, refresh page to start over)

Input

Map

Temp

Reduce

Output

All Events
